# Scooby Doo Data

Visualizing Scooby Doo Data in Neo4j with Vuetify and Chart.js

Backend:

Powershell: 

- cd backend
- python3.11.exe .\scoob-backend.py

- http://127.0.0.1:8080/totals
  Lists the totals of the Cypher command

Frontend:

Bash: npm run build && npm run preview

- http://127.0.0.1:4173

Neo4J: Open it & start
